# **App Name**: SheDebugs Portfolio

## Core Features:

- About Page: Showcase information about SheDebugs, highlighting its expertise in web, app, and software development.
- Services Page: Detail the services offered by SheDebugs: web development (Starter: R4000+, Business: R6000+, AI-Enhanced: R12000+), software development, and mobile app development.
- Contact Page: Enable users to contact SheDebugs through a contact form or displayed contact information, and offer live-chat powered by a gen-AI tool to answer any general inquiry about the services on offer.
- Portfolio Display: Showcase previous projects and customer testimonials.
- Payment Gateway Integration: Integrate a secure payment gateway to process payments for SheDebugs' services. Simulate by directing users to a dummy payment page.
- AI-Powered Service Recommendation: Based on client's business profile (described in natural language), suggest an appropriate service package, using a gen-AI tool.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke sophistication and innovation.
- Background color: Dark gray (#303030) for a modern and professional look.
- Accent color: Bright cyan (#00BCD4) to highlight important elements and call-to-action buttons.
- Body and headline font: 'Space Grotesk', sans-serif, for a modern, technical and easily readable style, especially suitable for headlines.
- Use sharp, modern icons to represent services and features.
- Ensure a clean and intuitive layout, focusing on easy navigation and clear information hierarchy.
- Subtle transitions and animations to enhance user experience and provide visual feedback.
